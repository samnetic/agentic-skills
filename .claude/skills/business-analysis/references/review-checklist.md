# Requirements Review Checklist

- [ ] Primary audience mode is explicit.
- [ ] Document is within target word budget.
- [ ] In-scope and out-of-scope are explicit.
- [ ] All FRs have testable acceptance criteria.
- [ ] All NFRs include numeric targets and validation method.
- [ ] Dependencies and risks have owners.
- [ ] Open questions are explicit and limited.
- [ ] Terminology is consistent across sections.
