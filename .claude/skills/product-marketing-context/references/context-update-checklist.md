# Context Update Checklist

Run this checklist before marking context as refreshed.

- [ ] ICP and non-ICP are still accurate.
- [ ] Competitive alternatives are current.
- [ ] Differentiation claims still map to product reality.
- [ ] Pricing/package descriptions match production offers.
- [ ] Core funnel metrics (if used in messaging) are current.
- [ ] Top objections reflect recent sales/support feedback.
- [ ] Last-updated date and owner are set.
