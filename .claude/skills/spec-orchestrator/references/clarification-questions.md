# Clarification Questions

Ask only what blocks quality gates.

## Core Questions

1. Who is the primary reader for this artifact?
2. What decision should this document enable?
3. What is explicitly out of scope?
4. What deadline or phase boundary applies?

## Conditional Questions

Use only when relevant:
- What compliance/security constraints apply?
- What current architecture or stack must be preserved?
- What baseline metric exists for success measurement?
- Is this a new capability or change to existing behavior?
