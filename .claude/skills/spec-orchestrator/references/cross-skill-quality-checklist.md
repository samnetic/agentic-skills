# Cross-Skill Quality Checklist

- [ ] Route and audience are explicit.
- [ ] Document type and output contract are explicit.
- [ ] Scope boundaries are explicit.
- [ ] FR and NFR are testable where relevant.
- [ ] Architecture options are compared where relevant.
- [ ] Metrics include numeric targets.
- [ ] Assumptions and open questions are visible.
- [ ] Final artifact is within target word budget.
- [ ] Next actions include owner and time horizon.
