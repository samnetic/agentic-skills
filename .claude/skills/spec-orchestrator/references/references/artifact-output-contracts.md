# Artifact Output Contracts

## PRD Contract

Required sections:
1. Problem and outcome
2. Audience and scope
3. Functional requirements + acceptance criteria
4. Non-functional requirements + measurable targets
5. Dependencies, risks, assumptions
6. Success metrics and next actions

## Requirements Spec Contract

Required sections:
1. Context and boundaries
2. FR catalog (ID + acceptance)
3. NFR catalog (metric + target + validation)
4. Decision log and open questions

## Architecture Design Contract

Required sections:
1. Context and constraints
2. Quality targets
3. Options compared
4. Recommendation and boundaries
5. Risk register
6. Rollout and validation plan
7. ADR delta

## ADR Pack Contract

Required sections:
1. Decision summary
2. ADR entries
3. Impact and migration considerations
