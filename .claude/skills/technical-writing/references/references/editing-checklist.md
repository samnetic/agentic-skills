# Editing Checklist

- [ ] Audience and purpose are explicit in first section.
- [ ] Exactly one document type is used.
- [ ] Headings are task/question oriented.
- [ ] Unnecessary background has been removed.
- [ ] Every claim has evidence or is marked assumption.
- [ ] Terms are consistent with project vocabulary.
- [ ] Word count is within mode budget.
- [ ] Final section tells reader what to do next.
