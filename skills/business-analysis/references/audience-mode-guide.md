# Audience Mode Guide

Choose one primary audience and optimize for that reader first.

| Mode | Primary Question | Required Sections | Avoid |
|---|---|---|---|
| `exec` | Should we fund this now? | Problem, impact, cost/risk, decision | Deep implementation detail |
| `product` | What are we building and why? | Users, scope, stories, success metrics | Low-level architecture |
| `engineering` | What do we need to build/test? | Requirement IDs, acceptance, edge cases, dependencies | Strategy-only prose |
| `architecture-input` | What constraints drive design? | NFRs, integrations, risks, scale assumptions | UI copy-level detail |

Length budgets are hard limits unless the user explicitly asks for full detail.
