# Prioritization Frameworks

## MoSCoW

- Must: release fails without it
- Should: important, but can ship later
- Could: optional improvement
- Won't: explicitly deferred

## RICE

`Score = (Reach x Impact x Confidence) / Effort`

Use when comparing roadmap candidates with different effort profiles.

## WSJF

`WSJF = Cost of Delay / Job Size`

Use when sequencing delivery in teams using weighted economics.

## Selection Rule

- Early discovery: MoSCoW
- Roadmap trade-offs: RICE
- Program-level sequencing: WSJF
